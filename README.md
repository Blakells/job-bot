# Job Bot 🤖

An AI-powered job application agent that:
- Builds a job profile from your resume + LinkedIn
- Scores job listings 0-100% against your profile
- Tailors your resume and cover letter per application
- Auto-fills and submits applications with human confirmation

## Stack
- **Hermes Agent** (Nous Research) — AI brain & orchestration
- **Browserbase** — Cloud browser infrastructure
- **Firecrawl** — Web scraping & data extraction
