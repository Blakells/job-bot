========================================
   AUTO_APPLY.PY - QUICK START GUIDE
========================================

LOCATION: scripts/auto_apply.py

RUN IT:
-------
Test mode:          python3 scripts/auto_apply.py --dry-run
Full automation:    python3 scripts/auto_apply.py --auto-submit
High quality only:  python3 scripts/auto_apply.py --min-score 80

WHAT IT DOES:
-------------
✅ Navigates multi-page job application forms
✅ Fills React-Select dropdowns (Greenhouse)
✅ Handles location autocomplete fields
✅ Uploads resume (PDF priority)
✅ Fills all text fields from profile
✅ Detects Continue vs Submit buttons
✅ Takes screenshots of each step

REQUIRED FILES:
---------------
uploads/Resume.pdf              (your resume - PDF or DOCX)
profiles/job_profile.json       (your profile data)
profiles/scored_jobs.json       (jobs to apply to)

PROFILE DATA USED:
------------------
personal.name                   → Full legal name
personal.email                  → Email fields
personal.phone                  → Phone: (843) 555-0192
personal.location               → Location: Florence, SC
personal.linkedin_url           → LinkedIn profile
years_of_experience             → Years experience: 4
extra_answers["How did you..."] → How did you hear: LinkedIn
extra_answers["Do you know..."] → Know anyone: No
work_authorization              → US work auth questions

NEW FEATURES:
-------------
1. File upload with PDF prioritization
2. Multi-page navigation (Continue buttons)
3. React-Select dropdowns (working code from test)
4. Location autocomplete (.type() not .fill())
5. All missing fields (LinkedIn, How did you hear, etc)

OUTPUTS:

outputs/screenshots/Company_01_start.png
outputs/screenshots/Company_02_filled.png

FLAGS:
------

WORKING DROPDOWN CODE:
----------------------
Exact pattern from test_dropdown_direct.py:
1. Find label by text
2. Get label's "for" attribute
3. Find combobox by ID
4. Click to open
5. Find option by text
6. Click option

This handles Greenhouse React-Select dropdowns perfectly.

LOCATION AUTOCOMPLETE:
----------------------
1. Click input field
2. Type city name (e.g. "Florence")
3. Wait 1.5s for suggestions
4. Click first visible option

Uses .type() not .fill() for React compatibility.

RESUME UPLOAD:
--------------
Searches uploads/ folder for:
1. *.pdf files (first)
2. *.docx files (fallback)

Returns absolute path for set_input_files()
